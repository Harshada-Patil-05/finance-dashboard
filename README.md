# 💰 Finance Dashboard

A modern, responsive **Finance Dashboard Web UI** built using React.js and Tailwind CSS.
It helps users track income, expenses, transactions, and view financial insights.


---

## ✨ Features

* Dashboard with charts (Line, Pie, Bar)
* Transactions management (Search + Filter)
* Insights page with analytics
* Role-based UI (Admin / Viewer)
* Fully responsive design
* Clean and modern UI

---

## 🛠️ Tech Stack

* React.js
* Tailwind CSS
* Recharts
* JavaScript

---


## 📂 Project Structure

src/
